# MonkHex

MonkHex is a production-ready Android discipline system built with Kotlin, Jetpack Compose, Firebase, and Gemini API integration.

## Features
- Single data-driven task recommendation algorithm
- Addiction category mapping (digital / mental / physical)
- XP, level, streak progression
- Commitment penalty and refund logic
- Firebase Auth, Firestore, FCM, Cloud Functions
- Premium dark futuristic UI in Compose

## Setup
1. Install JDK 17+.
2. Add `app/google-services.json`.
3. Create `local.properties` and set `GEMINI_API_KEY`.
4. Deploy functions from `firebase/functions`.
